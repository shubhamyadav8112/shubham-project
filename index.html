HTML (HyperText Markup Language) is the standard markup language used to create web pages. It provides the structure of a webpage using elements and tags.

Features of HTML

  • Easy to learn and use

  • What is HTML?

    Used to create web pages
  • Works with CSS and JavaScript
  • Platform independent

Basic Structure of HTML

Every HTML document contains tags such as html, head, title, and body. The content visible on the webpage is written inside the body tag.
